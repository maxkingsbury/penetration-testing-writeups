# Penetration Testing Writeups

This repository contains a collection of hands-on penetration testing engagements performed in self-hosted environments and platforms such as VulnHub.

The focus is on identifying, exploiting, and analyzing real-world vulnerabilities across Linux-based systems and web applications. Each writeup documents the full attack lifecycle from initial enumeration to privilege escalation.

---

## Skills Demonstrated

* Network Enumeration and Service Analysis (Nmap)
* Web Application Exploitation (WordPress, Drupal)
* Credential Attacks (brute force, credential reuse)
* Remote Code Execution (RCE)
* Privilege Escalation (SUID abuse, sudo misconfigurations, writable scripts)
* Post-Exploitation and Shell Stabilization
* System Enumeration using manual techniques and automation tools

---

## Methodology

Each engagement follows a structured approach:

1. **Enumeration**
   Identify open ports, services, and potential attack surfaces.

2. **Initial Access**
   Exploit vulnerabilities or weak credentials to gain a foothold.

3. **Privilege Escalation**
   Analyze the system for misconfigurations or exploitable paths to elevate privileges.

4. **Post-Exploitation**
   Stabilize shells, pivot users, and validate full system compromise.

## Writeups

* Privilege escalation techniques
